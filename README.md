# 全球天气时钟 - 边缘计算应用

## 项目简介
这是一个基于阿里云ESA Pages构建的边缘计算应用，能够根据访问者的IP地址自动定位，并实时显示当地的精确时间、天气信息和空气质量指数。

## 技术特色
### 🌍 边缘计算优势
- **低延迟**：所有逻辑在距离用户最近的边缘节点执行
- **智能缓存**：使用边缘KV存储缓存天气数据，减少API调用
- **全球覆盖**：依托阿里云全球3200+边缘节点

### 🛠️ 使用的ESA产品
1. **ESA Pages** - 应用托管与全球加速
2. **边缘函数** - 处理地理位置和天气API逻辑
3. **边缘KV存储** - 缓存天气数据（5分钟有效期）
4. **边缘缓存** - 静态资源全球加速

## 部署指南
### 前置要求
1. 阿里云账号（开通ESA服务）
2. GitHub账号
3. 大赛提供的10元代金券

### 部署步骤
1. **Fork本仓库**到你的GitHub账户
2. **登录阿里云ESA控制台**，进入Pages服务
3. **连接GitHub仓库**，选择本仓库
4. **配置构建设置**：
   - 构建命令：`无`（纯静态项目）
   - 输出目录：`/`
5. **配置路由规则**：
   - 路径：`/api/*` → 边缘函数
   - 函数路径：`edge-function/api/weather.js`
6. **部署并获取**.edgeapp.run域名
